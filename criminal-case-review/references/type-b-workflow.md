# 刑事阅卷 B类（委托类）专门工作流程

## 概述

本文档详细说明刑事阅卷B类（委托类）的专门工作流程，包括与A类（法律援助类）在流程上的差异，以及B类特有的步骤和注意事项。

---

## 一、B类阅卷前置准备

### 1.1 委托关系确认

在开始阅卷前，必须确认以下事项：

| 确认事项 | 说明 | 来源 |
|----------|------|------|
| 委托人身份 | 姓名、身份证号、与当事人关系 | 委托合同、家属陈述 |
| 委托事项 | 授权范围（阅卷、出庭、和解等） | 授权委托书 |
| 委托阶段 | 侦查/审查起诉/审判 | 委托合同 |
| 律师费支付 | 已付/待付/分期（不涉及具体金额） | 委托合同或律师确认 |
| 特殊授权 | 是否有权代为上诉、和解等 | 授权委托书 |

**重要提示**：
- 如果委托手续不完整，应在阅卷笔录中标注"委托手续待补充"
- 如果授权范围不明确，应主动与律师或委托人确认
- 不得擅自假设授权范围或编造委托事项

### 1.2 案件基础信息确认

B类案件中，通常需要额外确认：

| 信息项 | 说明 | 备注 |
|--------|------|------|
| 案号 | （202X）刑XX字第XX号 | 如已知，应在文书抬头中注明 |
| 当事人关系 | 委托人与当事人的关系 | 家属委托时需要确认 |
| 之前的律师工作 | 是否有前一阶段律师 | 如有，需要了解之前的工作情况 |

## 二、B类阅卷标准流程

### 2.1 表头部分（B类格式）

```
阅卷笔录

阅卷时间：YYYY年MM月DD日
提请审查起诉时间：YYYY年MM月DD日
阅卷地点：XX律师事务所
涉嫌罪名：XXXX
案号：（202X）刑XX字第XX号（如已知）

委托人：XXX，男/女，XXXX年XX月XX日出生，身份证号码：XXXXXXXX

阅卷内容
```

**与A类的差异**：
1. "阅卷地点"通常是律师事务所，而非法律援助中心
2. 可增加"案号"字段（如果已知）
3. "受援人"改为"委托人"

### 2.2 第一部分"基本情况"

**子标题1：基本信息**
```
1、基本信息
委托人XXX，男，XXXX年XX月XX日出生，身份证号码：XXXXXXXX，住址：XXXXXX，联系电话：XXXXXXXX。
```

**其他子标题**（与A类完全一致）：
- 2、身份情况
- 3、家庭情况
- 4、身体情况
- 5、前科情况
- 6、到案经过
- 7、强制措施
- 8、同案其他犯罪嫌疑人情况

### 2.3 第三部分"委托人供述与辩解"

**章节标题**：
```
三、委托人供述与辩解
```

**内容格式**（与A类完全一致）：
- 每次讯问笔录单独列出
- 必须包含完整元数据（时间、地点、讯问人、记录人、页码）
- 摘录部分直接复制原文
- 每次讯问笔录后必须跟【评价】

### 2.4 第十二部分"辩护思路"

**审查起诉阶段**：
```
（一）审查起诉阶段
1. 辩护人应当全面阅卷，核实证据的合法性、关联性、证明力。
2. 辩护人应当重点关注程序合法性，如讯问时间、地点、权利告知等。
3. 辩护人应当与委托人保持沟通，核实案件事实。
```

**与A类的差异**：
- 不使用"法律援助律师"表述
- 使用"辩护人"或"辩护律师"
- 不涉及法律援助的流程性内容

## 三、B类特有工作步骤

### 3.1 委托权限核对

**目的**：确认律师的授权范围，避免超出授权开展工作。

**核对内容**：
- 授权委托书的授权范围
- 是否有特殊授权（和解、上诉、撤诉等）
- 是否有限制性授权（仅限阅卷，不代表当事人出庭）

**记录方式**：
如发现授权不明确，应在阅卷笔录第一部分"基本情况"中标注：
> 委托授权范围：阅卷、提供法律咨询（详见授权委托书，存在限制性授权）

### 3.2 委托手续完善跟踪

**目的**：跟踪委托手续的完善情况，确保辩护工作不受影响。

**跟踪要点**：
- 委托合同是否签署
- 授权委托书是否完整
- 律师费支付情况（不涉及具体金额）

**记录方式**：
如委托手续不完整，应在文书抬头标注：
> 注：委托手续正在完善中，待补充。

### 3.3 家属沟通边界确认

**目的**：明确哪些信息可以与家属沟通，哪些信息只能与委托人本人沟通。

**确认内容**：
- 委托人是否允许与家属分享案件信息
- 是否有家属无权知悉的敏感信息
- 家属是否参与退赃、谅解等事项

**处理原则**：
- 保密义务优先：不得向家属透露委托人明确要求保密的内容
- 重大事项先与委托人本人沟通确认
- 家属意愿仅作参考，最终决定权在委托人本人

## 四、B类阅卷常见问题处理

### 问题1：委托手续不完整

**情形**：阅卷时发现委托合同或授权委托书尚未签署完整。

**处理方式**：
1. 在文书抬头标注"委托手续待补充"
2. 继续阅卷工作，但注意不超出合理授权范围
3. 在阅卷笔录中明确记录已知授权范围
4. 提醒律师尽快完善委托手续

**示例标注**：
> 注：委托手续正在完善中，授权范围待确认。当前已知：委托人家属签署委托合同，授权范围包含阅卷、提供法律咨询。

### 问题2：家属与委托人意愿不一致

**情形**：家属表达的诉求与委托人本人的意愿不一致。

**处理方式**：
1. 以委托人本人意愿为准
2. 在阅卷笔录中记录家属诉求，但标注"需与委托人本人确认"
3. 如果尚未会见委托人，应在辩护思路中列出需要核实的分歧点

**示例记录**：
> 家属反映：委托人希望认罪认罚，争取从轻处罚。但委托人本人尚未明确表态，需在会见中核实。

### 问题3：前一阶段律师工作未交接

**情形**：案件有前一阶段律师，但工作记录未交接。

**处理方式**：
1. 在阅卷笔录中记录前一阶段律师的工作阶段
2. 如果前一阶段有重要工作成果，应予以记录
3. 在辩护思路中考虑前一阶段工作的影响

**示例记录**：
> 前一阶段（侦查阶段）由XX律师事务所XX律师提供法律帮助，已提交法律意见书一份。

### 问题4：涉及退赃、谅解、和解

**情形**：案件涉及退赃、谅解、和解事项，家属希望律师协助。

**处理方式**：
1. 在阅卷笔录中记录涉案财物扣押情况
2. 在辩护思路中列出退赃、谅解的可能性
3. 明确标注退赃、谅解事项需与委托人和家属协商确认

**示例记录**：
> 涉案财物扣押情况：委托人涉案赃款XX万元已扣押（详见第八部分）。
> 辩护思路：如退赃、取得谅解，可能影响量刑。需与委托人和家属协商退赃意愿。

## 五、B类阅卷质量检查清单

### 5.1 前置准备检查

- [ ] 委托关系已确认（委托人身份、委托事项、委托阶段）
- [ ] 委托权限已核对（授权委托书范围）
- [ ] 案号已知（如有）
- [ ] 委托手续完善情况已记录

### 5.2 文书格式检查

- [ ] 文书抬头使用"委托人"而非"受援人"
- [ ] 阅卷地点为律师事务所（非法律援助中心）
- [ ] 案号已标注（如已知）
- [ ] 全文称谓统一（不混用"受援人"和"委托人"）

### 5.3 内容完整性检查

- [ ] 12部分结构完整
- [ ] 每次讯问笔录包含完整元数据
- [ ] 同案犯供述只摘录与委托人相关内容
- [ ] 每次讯问笔录后跟【评价】
- [ ] 辩护思路不涉及"法律援助"字样

### 5.4 B类特殊内容检查

- [ ] 不涉及"法律援助中心指派"等流程性内容
- [ ] 不涉及"法律援助无偿性质"等费用说明
- [ ] 委托手续如有缺失，已标注"委托手续待补充"
- [ ] 家属诉求如有记录，已标注需与委托人本人确认
- [ ] 退赃、谅解事项已记录，并标注需协商确认

### 5.5 通用规则检查

- [ ] 原件优先纠偏规则已遵守
- [ ] 供述归属校核规则已遵守
- [ ] 页码标注准确（不使用OCR行号）
- [ ] 落款完整（阅卷人、阅卷时间）

## 六、B类阅卷与后续工作的衔接

### 6.1 与会见笔录的衔接

**阅卷笔录 → 会见笔录**：
- 阅卷笔录中的争点清单（参与次数、角色分工、获利金额等）直接输入会见笔录
- 阅卷笔录中标注的"需与委托人本人确认"事项，在会见中重点核实
- 阅卷笔录中的家属诉求，在会见中向委托人求证

### 6.2 与辩护方案的衔接

**阅卷笔录 → 辩护方案**：
- 阅卷笔录中的证据分析直接支持辩护方案
- 阅卷笔录中的辩护思路展开为具体辩护方案
- 阅卷笔录中标注的退赃、谅解可能性，在辩护方案中具体化

### 6.3 与可视化图表的衔接

**阅卷笔录 → 可视化图表**：
- 阅卷笔录中的当事人信息 → 人物关系图谱
- 阅卷笔录中的犯罪事实 → 案件事实时间轴
- 阅卷笔录中的金额明细 → 委托人参与分析图、资金流向图
- 阅卷笔录中的程序节点 → 案件程序流程图

## 七、B类阅卷常见错误示例

### 错误1：称谓混淆

**错误示例**：
> 受援人XXX，男，XXXX年XX月XX日出生...

**正确示例**：
> 委托人XXX，男，XXXX年XX月XX日出生...

### 错误2：流程引用错误

**错误示例**：
> 辩护人应当根据法律援助法规定...

**正确示例**：
> 辩护人应当根据刑事诉讼法规定...

### 错误3：费用表述错误

**错误示例**：
> 本案不收取律师费用，由法律援助中心承担。

**正确示例**：
> 按委托合同约定，律师费用已支付。（或：委托手续待补充）

### 错误4：家属诉求与委托人意愿混淆

**错误示例**：
> 委托人希望认罪认罚，争取从轻处罚。（家属陈述，但委托人本人未确认）

**正确示例**：
> 家属反映委托人希望认罪认罚，但委托人本人尚未明确表态，需在会见中核实。
