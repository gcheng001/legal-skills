# criminal-investigation — 刑事侦查阶段文书生成

## 一、技能定位

本 skill 覆盖**侦查阶段**（拘留→审查逮捕→批准逮捕/不批准逮捕）律师核心工作，生成三类交付物：

| 编号 | 交付物 | 适用节点 |
|------|--------|----------|
| D1 | 会见提纲 + 当事人辅导要点 | 拘留后第一次会见前 |
| D2 | 取保候审申请书 + 不批准逮捕法律意见书 | 向公安/检察院争取变更强制措施 |
| D3 | 羁押必要性审查申请书 | 批捕后争取变更强制措施 |

**关键边界**：本 skill 的会见提纲是**无卷会见**——律师尚未看到任何案卷材料，仅凭拘留/逮捕通知书和家属陈述开展工作。与 `criminal-meeting`（基于阅卷笔录的有卷会见）截然不同，两者**不可混淆**。

---

## 二、全局护栏（强制执行，不可跳过）

1. **辩护贯穿全程**：侦查阶段的每一步工作都要为后续审查逮捕、审查起诉、庭审打基础。会见提纲的问题设计、意见书的论证方向，均须有庭审视角。
2. **系统独立立场**：不盲目顺从律师意见。发现论证漏洞、事实矛盾、法律依据不足时，主动提出，分析利弊，最终决策权在律师。
3. **当场解决问题**：每步发现的信息缺口或法律疑点，必须在当步讨论解决，不得带入下一步。
4. **完整记录讨论**：超过3轮的讨论自动记录到案件目录下的 `案件备忘录.md`。
5. **确认后才生成**：任何正式文书必须在对话中展示草稿、律师确认要点后，才生成 Word 文件，不得自动批量生成。

---

## 三、A/B 类型区分

每次启动时**询问一次**，确认后全程使用，不重复提问。

| 类型 | 当事人称谓 | 委托来源 | 律师称谓 |
|------|-----------|---------|---------|
| **A类（法律援助）** | 受援人 | 法律援助中心指派 | 法律援助律师 |
| **B类（委托）** | 委托人/当事人 | 当事人/家属委托 | 委托辩护人 |

---

## 四、执行流程（适用所有交付物）

```
Step 0  确认材料清单
         └─ 律师提供了哪些材料（拘留/逮捕通知书、家属陈述、当事人基本信息）
         └─ 缺失材料标【待补充】，不编造
Step 1  确认 A类 or B类（询问，不重复）
Step 2  在对话中展示内容草稿，供律师审阅
Step 3  等待律师确认（明确指令：可以/修改后确认/不生成）
Step 4  生成 Word 文件，告知完整路径
Step 5  用 code -r <文件路径> 打开文件
```

## 五、交付物 D1 — 会见提纲 + 当事人辅导要点

### 5.1 适用前提

- 律师**尚未阅卷**，仅凭通知书和家属提供的情况开展工作
- 如律师已有阅卷材料，应切换至 `criminal-meeting` skill

### 5.2 会见提纲结构

**封面**
```
会见提纲
嫌疑人：【姓名】
案由：【罪名，不确定则写"涉嫌……案"】
办案机关：【派出所/看守所名称】
强制措施：【拘留 / 逮捕】（附期限）
会见时间：【日期】
承办律师：【姓名】（A类注明法律援助中心；B类注明委托关系）
```

**第一部分：会见前准备清单**

律师进门前须核实的事项：

- [ ] 核实嫌疑人身份（姓名、身份证号、户籍地）
- [ ] 确认强制措施类型及期限（拘留：公安一般37天；逮捕后羁押期限）
- [ ] 确认是否已收到权利告知书（刑诉法第33条）
- [ ] 确认家属委托情况（A类：援助决定书是否送达；B类：委托书是否已签）
- [ ] 携带材料清单（律师证、律师事务所函、援助公函/委托书、空白笔记本）
- [ ] 了解看守所规定（是否限时、录音录像、手机禁入等）

**第二部分：会见重点问题提纲**

> 提示：问题按优先级排列，时间不足时优先完成前两板块

**【第一板块】基本情况确认**（5分钟内完成）

1. 是否收到《犯罪嫌疑人权利义务告知书》？内容是否理解？
2. 到案以来人身是否受到伤害？有无刑讯逼供情况（打、踢、长时间不让睡觉、不给吃饭、恐吓威胁）？
3. 现在身体状况如何？有无需要就医的情况？
4. 羁押场所条件如何？与其他人是否有矛盾冲突？
5. 是否有其他我需要了解的紧急情况？

**【第二板块】案情全面了解**（15-20分钟）

> 策略提示：先听当事人完整陈述，不打断，律师记录要点；再针对关键节点追问。

6. 从头说一下事情经过——什么时候、在哪里、和谁、发生了什么？（让当事人自行叙述，不要引导）
7. 你是怎么被带走的？（主动投案/被传唤/被抓？到案经过直接影响是否构成自首）
8. 到案后做了几次笔录？在哪里做的？大概说了什么内容？（初步判断供述一致性）
9. 有没有主动交代侦查机关尚未掌握的其他犯罪事实？（坦白/自首的区分）
10. 有没有揭发检举他人犯罪，或协助抓捕其他嫌疑人？（立功线索）
11. 涉案金额/数量/后果，你自己估算大概是多少？（量刑起点判断）
12. 被害人/损失方目前什么态度？有无赔偿可能？（谅解情节）
13. 同案人员情况——共几人、各自角色、是否均已到案？

**【第三板块】法律帮助**（10分钟）

14. 告知涉嫌罪名的基本构成要件（用白话解释，让当事人理解指控逻辑）
15. 讲解零口供 vs 如实供述的利弊分析：
    - 如实供述：可能获得坦白从宽、认罪认罚从宽；取保候审机会增加
    - 拒绝供述：合法权利，但侦查机关可能据其他证据定案；可能被认为态度差
    - **决策权在当事人，律师不代为决定**
16. 告知权利：会见权、阅卷权（审查起诉后）、申请取保候审权、申请回避权、申请鉴定权
17. 告知义务：不得串供、不得干扰证人、不得毁灭证据

**【第四板块】关键风险提示**（5分钟）

18. 不要被诱供、指供——侦查人员问"是不是这样""你当时是不是……"时，不要顺着说"是"；如实说自己知道的
19. 不要签自己不理解或与事实不符的笔录——可以要求重新记录，可以要求补充说明
20. 沉默权：对于不清楚的问题，可以说"我记不清了"或不回答；对于可能加重自己罪责的问题有不自证其罪的权利
21. 拒绝与同案人员接触或传话
22. 如被刑讯逼供，事后向律师、检察院、看守所驻所检察官反映

**第三部分：后续事项**

- 律师下次会见时间（预估）
- 家属需要配合的事项（退赃、联系被害方、准备谅解金等）
- 当事人如有新情况（如刑讯逼供、证人联系等），通过会见时告知律师

---

### 5.3 当事人辅导要点（律师会见时当场宣读/告知）

> 语言要求：大白话，当事人能听懂

```
【我是你的律师，以下几点你一定要记住】

第一，你有权利不回答警察的问题，这叫沉默权。
      但如果你有自首情节，主动如实交代可以从宽处理。具体怎么做，听我的建议。

第二，做笔录时，警察怎么问、你就原原本本说你知道的。
      不要顺着他们说"对""是的"，除非那真的是事实。
      如果笔录记的不对，你有权利要求改。改完了再签字。

第三，不管发生什么，不要跟同案的人传话、递纸条、打眼神。
      看守所里有监控。

第四，如果有人打你、威胁你、长时间不让你睡觉，
      这叫刑讯逼供，是违法的。你要记住时间、地点、是谁干的，告诉我。

第五，你家里的事我来联系。你只管好好配合该配合的，保护好自己。

有什么问题，等下次见面告诉我。
```

## 六、交付物 D2 — 取保候审申请书 + 不批准逮捕法律意见书

### 6.1 适用节点

- **取保候审申请书**：向公安机关或检察院申请变更强制措施，可在拘留后随时提出
- **不批准逮捕法律意见书**：在检察院审查逮捕期间（7日内）提交，争取检察院不批准逮捕

两份文书合并为一个 Word 文件，分上下两部分。

### 6.2 取保候审申请书

**法律依据**：《刑事诉讼法》第67条（取保候审条件）、第90条（申请程序）

**文书结构**：

```
取保候审申请书

申请人：【嫌疑人姓名】，【性别】，【出生日期】，【户籍地】
        现羁押于：【看守所名称】
        涉嫌罪名：【罪名】
辩护人：【律师姓名】，【律师事务所名称】
申请对象：【公安机关名称 / 检察院名称】

申请事项：
请依法对犯罪嫌疑人【姓名】变更强制措施为取保候审。

事实与理由：

一、【嫌疑人姓名】无前科劣迹，系初犯/偶犯
    【补充具体说明，无前科则写"此前无任何犯罪记录"；如实交代情况则写认罪态度】

二、具有固定住所和稳定工作，具备取保候审的客观条件
    【嫌疑人】现居住于【地址】，有固定住所；【工作单位/家庭情况，视具体情况补充】
    具备随传随到的条件。

三、认罪态度较好，主动配合侦查工作
    【如实供述/自首/坦白/退赃情况】

四、已积极退赃/赔偿，取得被害方谅解（如有）
    【被害方谅解情况，无则删除本节或写"正在积极联系被害方协商赔偿"】

五、同案犯均已到案，无串供、毁证可能
    【同案人员到案情况；如无同案则写"本案系独立犯罪，不存在串供可能"】

六、无逃跑、毁灭证据、干扰证人的可能
    【说明嫌疑人家庭、社会关系、经济状况等不具备潜逃条件的事实】

七、符合《刑事诉讼法》第67条取保候审条件
    根据《刑事诉讼法》第67条，可以取保候审的情形包括：
    （一）可能判处管制、拘役或者独立适用附加刑的；
    （二）可能判处有期徒刑以上刑罚，采取取保候审不致发生社会危险性的；
    （三）患有严重疾病、生活不能自理，怀孕或者正在哺乳自己婴儿的妇女，采取取保候审不致发生社会危险性的；
    （四）羁押期限届满，案件尚未办结，需要采取取保候审的。
    本案嫌疑人符合上述第（  ）项情形。【填写适用项并补充分析】

保证方案（择一）：
□ 保证金：申请人愿意缴纳保证金人民币【金额】元
□ 保证人：【保证人姓名】，【与嫌疑人关系】，【联系电话】，【住址】

综上，申请人符合取保候审法定条件，请贵院/贵局依法批准。

                            申请人辩护律师：【律师签名】
                            【律师事务所公章】
                            【日期】
```

**法律依据速查表**：

| 条文 | 核心内容 |
|------|---------|
| 刑诉法第67条 | 取保候审的四种适用情形 |
| 刑诉法第81条 | 逮捕的三项条件（证据条件+刑罚条件+社会危险性条件） |
| 刑诉法第90条 | 犯罪嫌疑人及其辩护人有权申请变更强制措施 |

### 6.3 不批准逮捕法律意见书

**适用时间**：检察院审查逮捕期间（一般为7日，重大案件可延长至37日）
**提交对象**：承办检察官（捕诉部门）

**文书结构**：

```
不批准逮捕法律意见书

编号：【律所编号，可空】

致：【检察院名称】捕诉部门

案件：【案由】——犯罪嫌疑人【姓名】不批准逮捕法律意见书
辩护人：【律师姓名】，【律师事务所名称】，联系电话：【电话】

尊敬的承办检察官：

本所接受【A类：法律援助中心指派/B类：当事人家属委托】，指派本所律师【姓名】担任犯罪嫌疑人【姓名】的辩护律师。根据《刑事诉讼法》第87条、第90条，现就本案提出不批准逮捕的法律意见，供贵院参考。

一、案件基本情况

犯罪嫌疑人【姓名】，【性别】，【出生日期】，涉嫌【罪名】，于【日期】被【公安机关】拘留，现羁押于【看守所】。辩护人已于【日期】会见当事人，了解基本案情如下：【简述当事人陈述的案件经过，2-3句话，注明系当事人陈述，尚未经证据核实】

二、本案不符合逮捕条件的具体论证

根据《刑事诉讼法》第81条，逮捕须同时满足证据条件、刑罚条件和社会危险性条件，三者缺一不可。本案存在以下不符合逮捕条件的情形：

（一）证据条件存疑【视案情选用，不适用则删除】
    【从证据收集的合法性、关键证据是否充分等角度论证；需谨慎，避免提前暴露辩护方向】

（二）刑罚条件：可能判处的刑罚较轻
    【根据罪名量刑幅度，论证本案情节较轻、可能判处较轻刑罚；需引用具体罪名量刑规定，须经元典MCP验证，标注【已验证】或【待元典验证】】

（三）社会危险性：不具有继续犯罪、逃跑、毁证等社会危险性
    根据最高人民检察院《人民检察院审查逮捕质量标准》，以下事实说明本案嫌疑人不具有社会危险性：
    1. 案发系偶发/临时起意，无预谋、无前科，再犯可能性低
    2. 嫌疑人有固定住所和家庭关系，无潜逃动机和能力
    3. 主要证据已固定（【已固定的证据类型】），嫌疑人无毁灭证据的条件
    4. 同案嫌疑人均已到案，无串供必要
    【据实补充，无依据的不写】

三、采取取保候审足以防止社会危险性

即便认定存在一定危险性，取保候审亦足以防止：
- 可设定固定住所限制，要求嫌疑人不得离开所在地市
- 可要求定期向公安机关报到
- 家属可作保证人，督促嫌疑人配合诉讼
- 缴纳保证金制度提供了经济约束

《刑事诉讼法》第67条第（二）项明确规定，可能判处有期徒刑以上刑罚，"采取取保候审不致发生社会危险性的"可以取保。本案符合这一条件。

四、嫌疑人具有从轻、减轻情节

【从以下情节中选取有据可查的，不编造】
- 自首（主动投案，如实供述主要犯罪事实）——《刑法》第67条第1款，可以从轻减轻处罚
- 坦白（如实供述侦查机关掌握的犯罪事实）——《刑法》第67条第3款，可以从轻处罚
- 立功（揭发他人犯罪/协助抓捕）——《刑法》第68条，可以从轻减轻处罚
- 退赃退赔，取得被害方谅解——《刑法》及相关司法解释中的酌定从宽情节
- 认罪认罚——《刑事诉讼法》第15条，从宽处理
- 初犯、偶犯、悔罪态度好——酌定从宽情节

上述情节表明，对嫌疑人继续羁押的必要性显著降低。

五、总结与请求

综合以上，本案嫌疑人不满足《刑事诉讼法》第81条规定的逮捕条件，且具有多项从宽情节，采取取保候审足以保证诉讼顺利进行。恳请贵院依法作出不批准逮捕决定，并对嫌疑人变更强制措施为取保候审。

如贵院需进一步了解案情，辩护人愿意随时配合。

此致
【检察院名称】

                            辩护律师：【律师签名】
                            【律师事务所名称】
                            【日期】
```

## 七、交付物 D3 — 羁押必要性审查申请书

### 7.1 适用节点

批准逮捕后，嫌疑人已被正式逮捕羁押。律师可依据《刑事诉讼法》第95条申请羁押必要性审查，请求检察院对羁押的必要性重新评估。

**提交对象**：负责本案的检察院（可以是逮捕决定检察院，也可以是侦查监督部门）
**关键时机**：批捕后侦查阶段新情况出现时（证据已固定、同案犯到案、健康恶化、家庭困难等）

### 7.2 羁押必要性审查申请书结构

```
羁押必要性审查申请书

致：【检察院名称】侦查监督部门（或诉讼监督部门）

申请人：犯罪嫌疑人【姓名】的辩护律师
辩护律师：【律师姓名】，【律师事务所名称】

案件：犯罪嫌疑人【姓名】涉嫌【罪名】案
当前状态：【日期】经贵院批准逮捕，现羁押于【看守所】

根据《中华人民共和国刑事诉讼法》第95条、《人民检察院办理羁押必要性审查案件规定》，现申请对犯罪嫌疑人【姓名】进行羁押必要性审查。

一、批捕后案件情况已发生重大变化

以下事实表明，维持逮捕强制措施已无必要：

（一）关键证据已固定，侦查主要工作已完成
    自【批捕日期】至今，侦查机关已完成：
    - 【已完成的侦查活动，如：讯问笔录已取、现场勘查已做、鉴定报告已出等】
    - 主要物证/书证均已扣押在案
    继续羁押不能达到固定证据的目的。

（二）同案犯罪嫌疑人均已到案，串供可能性已排除
    【同案人员到案情况；如系单独犯罪则修改此节为"本案系单独犯罪，不存在同案人员"】

（三）主要案件事实已查明，嫌疑人持续配合侦查
    【嫌疑人如实供述、配合调查的情况；侦查进度】

二、无继续羁押的必要性

（一）【新情况说明，从以下选项中选取适用的】

    选项A：侦查终结在即，刑事诉讼程序进入下一阶段
    本案侦查羁押期限届满/即将届满，案件即将进入审查起诉阶段，维持现有羁押状态对保障诉讼进行已无实质意义。

    选项B：家庭特殊困难（需提供证明材料）
    嫌疑人系家庭唯一经济来源，其家庭成员【关系：配偶/父母/未成年子女】目前面临【具体困难：疾病/生活无法自理/未成年子女无人照料】，已提交相关证明材料【列明附件】。

    选项C：身体状况明显恶化（需提供医学证明）
    嫌疑人现患【疾病名称】，病情【描述：加重/需要专业医疗条件】，看守所现有医疗条件不能满足治疗需要，已提交【医院诊断证明/在押健康状况证明，列明附件】。

三、变更强制措施不影响诉讼顺利进行

变更为取保候审后：
- 嫌疑人有固定住所，可设定居住限制，随传随到
- 家属可担任保证人，缴纳保证金，提供经济约束
- 侦查机关可对嫌疑人居所进行监控或定期汇报
- 嫌疑人已有认罪认罚意向，无对抗诉讼程序的动机

四、法律依据

- 《中华人民共和国刑事诉讼法》第95条：犯罪嫌疑人、被告人及其辩护人有权向人民检察院申请羁押必要性审查
- 《人民检察院办理羁押必要性审查案件规定》第16条：检察院对持续羁押可能有失公正的，应当提出释放或变更强制措施的建议
- 《刑事诉讼法》第81条：逮捕须有社会危险性，情况变化时应重新审查

五、申请事项

综合以上情况，申请贵院对犯罪嫌疑人【姓名】开展羁押必要性审查，并建议将强制措施变更为取保候审。

附件（如有）：
□ 家庭困难证明
□ 医疗诊断证明
□ 保证人情况说明
□ 保证金方案

此致
【检察院名称】

                            辩护律师：【律师签名】
                            【律师事务所名称】
                            【日期】
```

## 八、质量约束（强制执行）

1. **不编造事实**：所有陈述必须有材料依据；信息不足处标 `【待补充】`，不自行填充
2. **法条引用规范**：引用《刑事诉讼法》第67/81/90/95条时须完整标明条文内容；量刑规范（如《关于常见犯罪的量刑指导意见》中的具体数字）**必须通过元典MCP验证**，未验证前标注 `【待元典验证】`
3. **A/B类型一致**：全文当事人称谓、委托来源描述须与确认的类型一致，不得混用
4. **文书语言**：专业、精准，可直接提交给公安机关或检察院
5. **信息缺口处理**：发现关键信息缺失（如罪名不明、同案人数不清、强制措施期限不明），**当步解决**，询问律师后再继续

## 十一、升级版决策工具

> 本章节为侦查阶段提供两个决策辅助工具，嵌入既有流程使用，不替代上述任何交付物。
> **产出约束**：两个工具的产出同样遵守全局护栏——确认后才生成、基于材料不编造、引用法条及量刑规范须经元典MCP验证并标注 `【已验证】` / `【待元典验证】`。

### 11.1 工具一 — 逮捕必要性快速评估表

**使用时机**：在"评估逮捕必要性"环节使用，作为量化打分工具，辅助判断策略重心。

**三维评估**（源自《刑事诉讼法》第81条逮捕条件，三者缺一不可）：

| 评估维度 | 评估要点 | 本案情况 | 风险得分（低/中/高） |
|---------|---------|---------|------------------|
| 证据要件 | 是否有证据证明有犯罪事实 | 据会见信息填写 | |
| 刑罚要件 | 是否可能判处徒刑以上刑罚 | 据罪名情节判断 | |
| 社会危险性 | 是否可能实施新犯罪／危害国家、公共安全／毁灭伪造证据、干扰作证、串供／打击报复／自杀逃跑 | 逐项核对，引用具体事实 | |

**评分规则**：
- 三项中**任一为"高"** → 整体逮捕风险**高**
- **两项为"中"** → 整体风险**中**
- 其余情形 → 整体风险**低**

**策略联动**：

| 整体风险 | 策略重心 | 主攻文书 |
|---------|---------|---------|
| 低 | 强化取保条件（固定住所、保证金/保证人、谅解、退赃） | 主备《取保候审申请书》（D2上半部） |
| 中 / 高 | 立即转向"攻击逮捕要件"，重点论证无社会危险性或证据不足 | 主攻《不批准逮捕法律意见书》（D2下半部） |

> 提示：社会危险性是侦查阶段争夺的核心战场。打分时"社会危险性"一栏须逐项核对刑诉法第81条列举的五类情形，每项引用本案具体事实，不得笼统判断。

### 11.2 工具二 — 首次会见分析报告（会见后24小时内产出）

**定位**：D1 会见结束后的核心新交付物。会见是无卷阶段唯一的信息入口，须在**24小时内**趁记忆清晰固化分析结论，为后续逮捕必要性评估与文书撰写提供依据。

**报告结构**：

**第一部分 事实梳理**
- 以当事人视角清晰陈述案件经过：时间、地点、人物、行为、结果
- 注明系当事人单方陈述，尚未经证据核实
- 标注与家属陈述、通知书记载存在出入之处

**第二部分 争议焦点与辩点初判**
- 依五大核心原则（证据／程序／定性／量刑等），初判潜在辩点：
  - 证据辩点：关键证据是否充分、取证是否合法
  - 程序辩点：到案经过、笔录次数、有无刑讯逼供线索
  - 定性辩点：行为是否符合指控罪名构成要件、此罪与彼罪
  - 量刑辩点：自首/坦白/立功/退赃/谅解/初犯偶犯等从宽情节
- 引用罪名量刑规范处标注 `【待元典验证】`

**第三部分 风险评估与紧急程度评级**
- 套用 11.1【逮捕必要性快速评估表】给出逮捕风险评级（高／中／低）
- 明确下一步主攻方向：
  - 优先准备《取保候审申请书》，或
  - 重点挖掘无罪/罪轻证据、攻击逮捕要件
- 标注紧急程度（如审查逮捕7日期限临近，须优先提交《不批准逮捕法律意见书》）

**第四部分 后续待办清单**
- 需家属配合调取的证据（病历、聊天记录、转账凭证、证人联系方式等）
- 需向办案机关了解的信息（强制措施期限、是否已移送审查逮捕、承办人联系方式）
- 下次会见预估时间与重点

**附：辅助证明材料清单（指导家属同步准备）**

为取保候审/不批准逮捕的附件做前置准备，提示家属同步收集：
- 身份证、户口本（证明身份及固定住所）
- 工作证明、劳动合同、社保记录（证明稳定工作、随传随到条件）
- 病历、医院诊断证明（证明患病/生活不能自理等取保情形）
- 家庭特殊困难证明（唯一经济来源、需抚养未成年子女/赡养老人等）
- 退赃退赔凭证、被害方谅解书（如有）
- 保证人情况说明或保证金方案

> 清单按个案需要勾选，缺失项标 `【待家属补充】`，不替家属臆造材料。
