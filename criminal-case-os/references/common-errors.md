# 常见错误清单

> 输出前自检，避免常见错误。

---

## 工作流错误

- [ ] **一次性生成全部内容**：阅卷笔录必须逐部分撰写，逐部分确认，不得擅自跳到下一部分
- [ ] **擅自继续写下一部分**：写完一部分后必须等用户确认，不得自行续写
- [ ] **类型选择错误**：未询问案件类型或混淆A类（法律援助）和B类（委托）
- [ ] **无卷会见与阅卷后会见混用**：侦查阶段的无卷会见与一审阶段的阅卷后会见严格区分

---

## 内容性错误

- [ ] 供述归属错误（张三的供述写成李三）
- [ ] 讯问字段缺失（时间/地点/人员/页码）
- [ ] 关键摘录不完整
- [ ] **缺少【评价】**：每次讯问笔录后必须跟评价
- [ ] 建议性语言过强，不符法律文书语体
- [ ] 模板提示语未删除
- [ ] **AI腔未去除**：输出前未执行去AI味自检（见灵魂文件第6条）
- [ ] 编造案卷中不存在的内容
- [ ] 量刑建议与情节认定脱节

---

## 格式错误

- [ ] **字体错误**：Word导出时全文应使用仿宋_GB2312
- [ ] **字号不分层级**：大标题、子标题、正文必须区分字号
- [ ] **遗漏落款**：文档末尾必须有"阅卷人"和"阅卷时间"
- [ ] **编号格式混乱**：章节子标题应使用阿拉伯数字

---

## 安全错误

- [ ] **脱敏不足**：对外材料泄露辩护策略底牌
- [ ] **过度承诺**：用确定性语气承诺无罪辩护结果
- [ ] **未标注不确定性**：信息不足时未明确说明

---

## 自检流程

### 输出前必过清单

1. 第一句是否已进入结论或判断？（不是开场废话）
2. 有没有"我已经看过""下面我来分析"？
3. 有没有"核心判断""先给结论"套娃标题？
4. 有没有"不是X，而是Y"口号句？
5. 有没有对律师的夸奖、安抚、鼓劲？
6. 抽象词是否落到具体事实、法条、证据、动作？
7. 最后一句是否实质内容？（收尾姿态句删掉）
8. `真正/真的` 出现几次？超过2次就删。

### 用词频率红线

| 词汇/句式 | 限制 |
|-----------|------|
| `真正/真的/本质上/归根到底` | 全文最多2次 |
| `真正的X，是Y` 升华句 | 整篇最多1次 |
| `形成/长成/沉淀成` 动作名词化 | 全文最多3次 |
| 三并列/四并列节奏句 | 整篇最多2次 |
