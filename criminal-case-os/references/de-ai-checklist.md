# 去AI味自检清单

> 所有文书、分析、报告在输出给律师前，必须先经过去AI味处理。

---

## 输出前自检清单（每次输出必过）

- [ ] 第一句是否已进入结论或判断？（不是开场废话）
- [ ] 有没有"我已经看过""下面我来分析"？
- [ ] 有没有"核心判断""先给结论"套娃标题？
- [ ] 有没有"不是X，而是Y"口号句？
- [ ] 有没有对律师的夸奖、安抚、鼓劲？
- [ ] 抽象词是否落到具体事实、法条、证据、动作？
- [ ] 最后一句是否实质内容？（收尾姿态句删掉）
- [ ] `真正/真的` 出现几次？超过2次就删。

---

## 高频禁区速查

| 类型 | 禁止 | 推荐 |
|------|------|------|
| 开场废话 | "我已经把案卷材料完整读过了。下面我基于阅卷内容做一个分析。" | "这个案子有两个核心辩点：主观明知和金额认定。" |
| 升华句式 | "真正的辩护，不是找出几个程序瑕疵，而是从根本上瓦解控方证据体系。" | "辩护的核心是动摇控方证据链。" |
| 教练腔鼓劲腔 | "你已经有了处理复杂刑事案件的脑子，但你的阅卷深度还配不上你现在的辩护层级。" | "阅卷深度不够，重点证据需要逐笔核对。" |
| 对比句式造势 | "不是不懂法条，而是底层动作还不够硬。" | "法条理解没问题，短板在证据分析的细致程度。" |
| 套娃标题 | "## 核心判断"、"## 先给结论"、"## 一句话总结" | 前文说完就结束，不再另起一轮重复总结。 |
| 结尾姿态句 | "方向已经明确"、"未来可期"、"拭目以待"、"这只是开始" | 写到结论落地就停。 |
| 工整排比 | "既要说明主观故意，又要说明客观行为，还要说明因果关系" | 打破工整结构，改为陈述句。 |
| 程式化连接词堆叠 | "首先…其次…此外…综上所述" | 用 `所以`、`因为`、`不过`、`其实` 自然连接。 |

## 用词频率红线

| 词汇/句式 | 限制 |
|-----------|------|
| `真正/真的/本质上/归根到底` | 全文最多2次 |
| `真正的X，是Y` 升华句 | 整篇最多1次 |
| `形成/长成/沉淀成` 动作名词化 | 全文最多3次 |
| 三并列/四并列节奏句 | 整篇最多2次 |
