# 变更日志

## [2.0.0] - 2026-05-19

### 重大变更 🔀
- **精简为五步**：下载→保存→飞书（仅传票/开庭），去掉文件总结和PDF后处理
- **下载路径固定**：`~/Desktop/文书下载/{案号}/`，不扫描案件目录
- **原样保存**：保留API原始文件名，同名异内容时追加 `_2` 后缀防止覆盖
- **只读关键文书**：仅提取传票和开庭/出庭通知书写飞书，其他文书一概跳过
- **强化触发**：description 明确 TRIGGER 条件，避免绕路尝试其他工具

### 移除 ✂️
- 文件内容总结步骤（第四步，较慢）
- PDF后处理（合并、重命名、页面标准化）
- 案件目录扫描匹配逻辑

## [1.4.0] - 2026-05-19

### 新增 🎉
- **文件内容自动总结功能**：下载后自动读取每份文书并生成结构化总结
  - 传票/出庭通知书：案号、法院、当事人、时间、地点
  - 起诉状/公诉书：诉讼请求、事实与理由概要
  - 判决书/裁定书：判决结果、上诉期、履行期
  - 通知书类：核心期限、法律后果
  - 其他文书：内容摘要

- **结构化输出格式**：使用表格和边框美化总结报告，便于快速浏览

### 变更 🔄
- 工作流从六步扩展为七步
  - 第四步：文件内容总结（新增）
  - 原第四步（归档）→ 第五步
  - 原第五步（飞书）→ 第六步
  - 原第六步（后处理）→ 第七步

### 改进 ✨
- 用户下载文书后立即看到内容总结，无需打开 PDF
- 关键信息高亮提醒（时间、期限、判决结果等）

## [1.3.0] - 2026-05-19

### 新增 🎉
- **关键日期信息提取功能**：自动提取文书中的关键时间信息
  - 传票/开庭通知书：开庭时间、地点
  - 举证通知书：举证期限
  - 缴费通知书：缴费期限
  - 取证通知书：取证时间、地点
  - 判决书/裁定书：上诉期、履行期
  - 调解书：生效时间、履行期限
  - 执行通知书：履行期限、报告财产期限

- **飞书多维表格集成**：自动将关键日期信息写入飞书多维表格
  - 支持自定义字段映射
  - 配置文件：`config/feishu-config.json`
  - 使用 lark-cli 进行 API 调用

### 实测经验 ✅
- **飞书 CLI 写入正确姿势**（2026-05-19实测成功）：
  - ✅ 使用【字段名】（中文）而非 field_id
  - ✅ 单选字段直接用选项名（"高"/"中"/"低"）
  - ✅ 日期时间字段传毫秒时间戳：`$(date -j -f "%Y-%m-%d %H:%M" "时间" "+%s")000`
  - ❌ 错误用法：使用 field_id 会报错 FieldNameNotFound

### 变更 🔄
- 工作流从四步扩展为六步
  - 新增第五步：关键信息提取与飞书写入
  - 原第五步（PDF后处理）改为第六步

### 文档 📝
- 更新功能概述和描述
- 新增飞书配置说明（含实测经验）
- 新增文书类型识别与信息提取规则表格
