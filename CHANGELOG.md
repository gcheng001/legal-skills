# 变更日志

## [1.3.0] - 2026-05-19

### 新增 🎉
- **关键日期信息提取功能**：自动提取文书中的关键时间信息
  - 传票/开庭通知书：开庭时间、地点
  - 举证通知书：举证期限
  - 缴费通知书：缴费期限
  - 取证通知书：取证时间、地点
  - 判决书/裁定书：上诉期、履行期
  - 调解书：生效时间、履行期限
  - 执行通知书：履行期限、报告财产期限

- **飞书多维表格集成**：自动将关键日期信息写入飞书多维表格
  - 支持自定义字段映射
  - 配置文件：`config/feishu-config.json`
  - 使用 lark-cli 进行 API 调用

### 实测经验 ✅
- **飞书 CLI 写入正确姿势**（2026-05-19实测成功）：
  - ✅ 使用【字段名】（中文）而非 field_id
  - ✅ 单选字段直接用选项名（"高"/"中"/"低"）
  - ✅ 日期时间字段传毫秒时间戳：`$(date -j -f "%Y-%m-%d %H:%M" "时间" "+%s")000`
  - ❌ 错误用法：使用 field_id 会报错 FieldNameNotFound

### 变更 🔄
- 工作流从四步扩展为六步
  - 新增第五步：关键信息提取与飞书写入
  - 原第五步（PDF后处理）改为第六步

### 文档 📝
- 更新功能概述和描述
- 新增飞书配置说明（含实测经验）
- 新增文书类型识别与信息提取规则表格
- 新增字段映射参考表
