# criminal-trial-examination — 庭审发问与举证质证

## 定位

一审阶段庭前会议 + 庭审实战文书生成工具。覆盖从庭前会议申请策略到逐份质证意见的全套输出。

**对应交付物**：交付物7（庭前会议提纲 + 排非申请书）+ 交付物8（庭审发问与举证质证提纲）

**前置依赖**：必须已完成刑事阅卷笔录（criminal-case-review 输出）。严禁脱离阅卷材料空泛生成。

---

## 全局护栏（继承自刑事案件OS）

1. **辩护贯穿全程**：每项输出均以有效辩护为目标，不做中立陈述
2. **系统独立立场**：AI 不偏向公诉方，所有分析从辩护角度出发
3. **当场解决问题**：庭上口述版须可直接宣读，无需临场改写
4. **完整记录讨论**：律师审核意见逐条记录，写入 LOG.md
5. **确认后才生成**：草稿经律师逐份审核确认，方可生成 Word 文件

**A/B 类型区分**：
- A 类（法律援助）：文书称谓用"被告人姓名 + 辩护人"
- B 类（委托）：文书称谓用"辩护人受托代理……"

---

## 执行流程

```
步骤1：读取阅卷笔录
  → 确认案件类型（A/B）
  → 提取：被告人供述变化、证据瑕疵线索、关键时间节点、涉案物证/书证清单

步骤2：用户选择模块
  → [仅庭前会议] → 执行模块A
  → [仅庭审发问质证] → 执行模块B
  → [全套] → A + B 顺序执行

步骤3：生成草稿 → 律师逐份审核

步骤4：确认后生成 Word 文件
  → 调用 md2word 或 docx skill
  → 输出路径：案件目录/intermediate/庭审文书-[案号]-[日期].docx
```

  □ 其他：

```

**非法证据速查表**（供快速判断适用条款）：

| 证据类型 | 绝对排除情形 | 裁量排除情形 |
|---------|------------|------------|
| 被告人供述 | 刑讯逼供、暴力威胁取得 | 未按规定全程录音录像；讯问地点违规 |
| 证人证言/被害人陈述 | 暴力、威胁、引诱取得 | 询问程序违规但可补正说明 |
| 物证、书证 | — | 收集程序违法且无法补正或合理解释 |
| 鉴定意见 | 鉴定机构/人员不具备资质 | 鉴定程序不符合国家标准规定 |
| 电子数据 | 无法保证真实性和原始性 | 收集程序有瑕疵但可补正 |
| 勘验、检查笔录 | 程序严重违法 | 未通知当事人参加且无法解释 |

## 模块B：庭审发问 + 举证质证提纲

### B1 公诉人可能发问的问题（预判 + 应对方案）

基于阅卷材料中被告人供述内容，预判公诉人发问方向：

| 公诉人可能问 | 发问意图 | 被告人应对方案 |
|-------------|---------|--------------|
| "你在侦查阶段说的是不是实话？" | 锁定口供一致性，堵死翻供空间 | "我在侦查阶段说过的话，以今天庭审说的为准。我今天说的都是实话。" |
| "你当时为什么那么做？" | 诱导自认主观故意 | "当时的情况是……（客观描述行为经过，不主动评价主观心态）" |
| "你知不知道那是违法的？" | 证明主观明知要件 | "我当时没有意识到……因为……" |
| "你认不认罪？" | 争取认罪认罚情节 | [由辩护策略决定——与律师确认后填写] |
| [根据具体案情补充] | | |

**律师提示**：被告人应对方案须在庭前充分沟通，避免当庭超出预案。

---

### B2 辩护人对被告人的发问提纲

**发问三原则**：
1. 每个问题服务于一个明确辩护目标（不问无效问题）
2. 优先开放式问题（让被告人展开有利陈述）
3. 封闭式问题仅用于固定已有证据

**三类发问模板**：

**【关于到案经过】** — 服务自首/立功认定

```
问1：案发后，你是如何到案的？（由谁带走/主动投案/被抓获）
问2：到案之前，你是否向家人或他人说起过这件事？说了什么？
问3：（如主动投案）你当时是什么想法？
问4：到案后，你是否如实回答了侦查人员的问题？
```

**【关于主观心态】** — 服务无罪/罪轻认定

```
问1：案发当时，你对[涉案事项]了解多少？
问2：你当时是否知道[构成要件事实]？
问3：你的目的/想法是什么？（引导陈述主观上无故意/过失）
问4：如果你知道[后果]，你会怎么做？
```

**【关于量刑情节】** — 服务量刑

```
问1：对于给[被害人/社会]造成的影响，你有什么想说的？
问2：你现在是否认识到自己行为的错误？
问3：[如有赔偿]你或你的家人是否已经做过赔偿？赔了多少？
问4：关于你的家庭情况，能简要介绍一下吗？（子女/老人/重大疾病）
```

---

### B3 对证人/鉴定人的交叉询问提纲

**适用前提**：公诉方已申请证人/鉴定人出庭，或辩护方申请获准。

**问题树结构**（逐人展开）：

```
证人/鉴定人：[姓名/单位]

主攻方向：[可信性/专业性/程序合规性/与其他证据矛盾]

封闭式主问题1：[只能答是/否]
  → 如果答"是"：追问1-1 → 追问1-2（锁定矛盾，引出对我方有利事实）
  → 如果答"否"：追问1-3（与在案证据X产生矛盾，质疑可信性）

封闭式主问题2：[只能答是/否]
  → ……
```

**对鉴定人专项质疑方向**：

| 质疑维度 | 具体问题方向 |
|---------|------------|
| 鉴定资质 | 机构是否具有相应鉴定资质？鉴定人是否持有执业证书？ |
| 检材完整性 | 检材来源是否有封存记录？提取过程是否有录像？ |
| 鉴定程序 | 是否按照国家标准/行业规范操作？偏差说明在哪里？ |
| 鉴定方法 | 采用的方法是否经过验证？是否为领域内通行方法？ |
| 鉴定意见确定性 | 结论是"确定""可能"还是"不排除"？概率是多少？ |

  证明目的：[该证据要证明的具体事实]
  出示方式：宣读 / 展示（PPT/实物）/ 播放（视频/录音）

   证明[支持哪个辩护观点]。」

```

### B5 逐份质证意见（对公诉方举证）

**格式说明**：每份证据生成书面版 + 庭上口述版两个版本。

---

**证据X：[证据名称]**

**书面质证意见**：

一、关于真实性
[是否有伪造/变造迹象；原件核对情况；来源合法性]

二、关于合法性
[取证程序是否合规；是否存在非法取证情形；载体形式是否符合要求]

三、关于关联性
[该证据与待证事实是否具有实质性关联；能否独立证明控方主张]

四、综合意见
[建议：不予采信 / 存疑 / 降低证明力]

---

**庭上口述版（可直接宣读）**：

```
「审判长，辩护人对这份[证据名称]发表质证意见。

关于真实性：[一句话结论]。

关于合法性：[一句话结论，如有程序问题直接点出]。

关于关联性：[一句话结论]。

综合以上，辩护人认为这份证据[不应作为定案依据 / 证明力存疑 / 应与其他证据相互印证]。」
```

## 非法证据排除申请书法律依据快查

- 《刑事诉讼法》第56条：非法方法收集的证据应当予以排除
- 《刑事诉讼法》第57条：排非程序启动条件与举证责任分配
- 《关于办理刑事案件严格排除非法证据若干问题的规定》（2017）：
  - 第2条：非法取得被告人供述的认定
  - 第3条：威胁、引诱、欺骗取证的界定
  - 第5条：录音录像瑕疵的处理
  - 第14条：法庭质疑程序
- 《最高人民法院关于适用〈刑事诉讼法〉的解释》第95-101条：非法证据排除程序

### 工具1：质证指令卡（正反双面卡片）

为每一份关键公诉证据制作一张正反双面卡片，庭审中可即时调用、即时宣读。**正面用于快速判断，背面用于直接开口**。

**卡片制作规则**：
- 仅对"关键证据"制卡（定罪量刑核心证据、存在三性瑕疵的证据），不对无争议证据浪费卡片
- 正面三性评分：真实性 / 合法性 / 关联性，逐项用 ✅（无异议）⚠️（存疑）❌（有重大问题）标注
- 页码必须是真实卷宗页码（如"诉卷第A-B页"），禁止OCR行号
- 背面口头质证版严格控制在 3 句话以内，确保庭上一口气读完

**标准格式**：

```
【质证指令卡·证X】

正面（快速判断）：
  证据名称：XXX（诉卷第A-B页）
  三性评分：真实性⚠️ / 合法性❌ / 关联性✅
  核心问题：取证程序违法（无见证人签字）

背面（口头质证版，≤3句）：
  "审判长，对这份证据合法性提出异议：一、取证时无见证人；
   二、与XX证据矛盾。请求不予采信。"
```

**使用提示**：开庭前按公诉人举证顺序排列卡片，举证到哪张抽哪张；正面快速比对评分定基调，翻到背面直接宣读。

### 工具2：当事人庭审应答"三板斧"法则

辅导被告人当庭应答的核心法则，庭前演练时反复强调，确保被告人形成肌肉记忆：

1. **听清问题**——不抢答，先确认公诉人问的到底是什么，听不清可请求重复
2. **简短事实回答**——只答客观事实，不展开、不解释动机
3. **不猜测、不延伸**——不知道就说"不知道/记不清了"，绝不主动评价自己的主观心态（是否"明知""故意")

**配套交付物：《高危问题清单》**

把公诉人可能提出的敏感、诱导性问题逐条列出，开庭前对被告人**重点演练，至少两轮**。

| 高危问题 | 风险点（公诉人意图） | 标准应答示范 | 演练情况 |
|---------|------------------|------------|---------|
| "你当时知不知道这是违法的？" | 诱导自认主观明知 | "我当时没有意识到……" | □第一轮 □第二轮 |
| "你为什么这么做？" | 诱导自认主观故意 | "当时的情况是……（只述客观经过）" | □第一轮 □第二轮 |
| "侦查阶段说的是不是实话？" | 锁定口供、堵翻供 | "以今天庭审陈述为准，今天说的都是实话" | □第一轮 □第二轮 |
| [据具体案情补充] | | | □第一轮 □第二轮 |

**律师提示**：高危问题清单须结合本案阅卷笔录中被告人供述的薄弱点定制，演练后记录被告人临场反应，针对仍易失误的问题追加演练。

### 工具3："总-分-总"质证法（火力集中）

解决"质证意见散乱、各打各的、未集中攻击证据体系核心"的问题。整场质证须围绕**证据体系最薄弱处**形成合力：

- **总（开庭陈述/质证总纲）**：用一句话点明公诉方证据体系的根本缺陷（如"本案定罪完全依赖口供，缺乏客观证据印证"），立下总纲
- **分（逐份质证）**：每质证一份证据，都把该证据的瑕疵**关联回这个根本缺陷**——不是孤立指出"这份有问题"，而是说明"这份证据的瑕疵进一步暴露了证据体系XX的根本缺陷"
- **总（法庭辩论）**：综合全部质证，总结证据不足，呼应开篇根本缺陷，形成闭环

**根本缺陷定位表**（开庭前先填，定下全场主攻方向）：

| 候选缺陷 | 是否成立 | 支撑证据（卷宗页码） | 是否定为总纲 |
|---------|---------|------------------|------------|
| 核心证据依赖单一口供，无客观印证 | | | |
| 关键物证取证程序违法 | | | |
| 鉴定意见不具备资质/程序违法 | | | |
| 证据链存在无法弥补的断点 | | | |

**使用提示**：每张质证指令卡（工具1）背面口述意见，结尾尽量回扣总纲一句，确保"分"始终服务于"总"，火力不分散。
